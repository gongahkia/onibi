Branding

Onibi is not affiliated with the Ghostty terminal emulator project.

Onibi may use xterm-ghostty terminfo for terminal compatibility and can launch Ghostty for macOS handover. The Ghostty name is a factual technical reference, not endorsement, sponsorship, affiliation, or a claim that Onibi visually matches or controls Ghostty.

Usage Boundaries

  • Use "Onibi" only for this project, CLI, daemon, web cockpit, docs, and release artifacts.
  • Do not use the Ghostty name, logo, or trade dress as Onibi branding.
  • Do not imply Ghostty endorses, sponsors, ships, or maintains Onibi.
  • Keep Ghostty references technical: macOS handover and terminfo compatibility only.
  • Link this page from prominent Ghostty references in Onibi docs.
  • Link Ghostty references to https://ghostty.org when the terminal emulator project is being named.
