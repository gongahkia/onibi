Onibi

Onibi is a single-user local coding-agent host with a paired phone web cockpit.

It runs a managed terminal session locally, serves it over HTTPS/WebSocket, and presents Claude Code approvals to the paired owner browser. Ghostty handover is macOS-only.

Start

brew install gongahkia/onibi/onibi
onibi up --transport=lan
